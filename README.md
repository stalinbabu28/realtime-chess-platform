# Real-Time Chess Platform

## Overview

A backend-heavy real-time multiplayer chess platform built using Java, Spring Boot, PostgreSQL, WebSockets, and JWT authentication.

The project focuses on:

- real-time gameplay
- backend architecture
- authentication
- matchmaking
- database design
- scalable layered backend systems

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React |
| Backend | Spring Boot |
| Database | PostgreSQL |
| Real-Time | WebSockets |
| Authentication | JWT |
| ORM | Hibernate/JPA |

## Planned Features

- User authentication
- Real-time gameplay
- Matchmaking
- Match history
- ELO rating system
- Timers
- WebSocket synchronization
- Leaderboards

## Architecture

```
Frontend
    ↓
REST APIs + WebSockets
    ↓
Spring Boot Backend
    ↓
PostgreSQL
```
